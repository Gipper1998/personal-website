FSI-NASA Capstone

2023

I was a part of an amazing Capstone team with it's main objecting being to dig a flag walled and floored trench from the top of a lunar crater to the bottom. This will be done using a RASSOR rover by adding a flywheel based digging tool that can be mounted to perform the operation at a fifteen degree angle. Throughout the project there was autonomous system being developed, the flywheel contraception, and developing the circuit model for moving around the terrain using LiDAR sensors and DC motors. The RASSOR pictured was provided by the Florida Space Institute (FSI) whom works in conjunction with the National Aeronautics and Space Administration (NASA).

Unity Game (Documents Duly)

2024

During the Spring of 2024, I was part of a team collaborating to make a Virtual Reality game inspired by Papers Please to create "Documents Dully." The main functionality is to verify passports to see if they are good to enter into a new country, or is denied entry. It was developed using Unity version 2021.3.16f1 with OpenXR tools enabled with the Oculus Rift S headset for testing. Mixamo was also used to make the characters come to life. My Github page includes the Unity project as a contributor for anyone to take a look, for game play footage please contact me for more information.
